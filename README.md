# 🧮 Infinite Calculator Pro

A modern, responsive calculator and converter featuring a beautiful glassmorphism UI. It includes real-time calculation previews, session history, and a seamless light/dark mode toggle.

## ✨ Features

* **Multiple Operations:** Addition, Subtraction, Multiplication, Division.
* **Unit Converters:** Celsius to Fahrenheit, Kilometers to Miles.
* **Volume Calculator:** Calculate the volume of Cubes, Cuboids, and Cylinders dynamically.
* **Live Preview:** See your results instantly as you type.
* **Calculation History:** Keeps track of your recent calculations during the session.
* **Theme Switcher:** Toggle between Dark Navy/Purple and clean Light Mode.
* **Clipboard Support:** One-click copy for your results with toast notifications.
* **Glassmorphism UI:** Modern, translucent cards with neon accents and smooth animations.
